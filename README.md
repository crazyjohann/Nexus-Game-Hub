🎮 Nexus Game Hub

Welcome to the Nexus Game Hub, the ultimate, highly customizable, browser-based gaming platform. Designed with a sleek, glassmorphic UI and neon aesthetics, this hub provides a premium, seamless way to play hundreds of web games without popups or clunky interfaces.

✨ Features

Massive Game Library: 150+ carefully categorized base games, ranging from high-octane driving to casual puzzle games.

🔒 The Vault: A secret, hidden collection of premium games. Only accessible to those who know the secret trigger.

🥷 Stealth Mode (Panic Button): Instantly hide the game hub by pressing the ~ (tilde) key. The screen instantly transforms into a fake, fully-functioning Google search page. Press ~ again to return to the action.

🎨 Ultimate Customization:

12 distinct accent color themes (Cyber Deep, Neon Magenta, Electric Ocean, etc.).

3 unique typography pairings.

Sleek Dark Mode and clean Light Mode toggles.

🧑‍🚀 Advanced Player Profiles:

Set a custom username that links to game requests and reports.

Choose from preset avatars, paste direct image URLs, or use the Google Images Assistant to seamlessly drag-and-drop or paste custom icons. Includes built-in zoom and pan tools!

🎬 Cinematic Game Launcher: Games open in a distraction-free, blur-backed modal complete with specific controls, pro tips, and fullscreen support.

📡 Live Requests & Reporting: Built-in forms powered by Web3Forms allow users to request new games or report broken ones directly to the admin.

🚀 Getting Started

Nexus Game Hub is built completely in standard web languages (Vanilla JavaScript, HTML5, and Tailwind CSS via CDN). No complex servers or build steps are required!

Download/Clone the repository.

Open index.html in any modern web browser (Chrome, Edge, Firefox, Safari).

Enter the Hub Password: The default entry password is nexusgames.

🤫 Secret Controls & Admin Info

To get the most out of the hub, keep these built-in secrets in mind:

Entering the Vault: While on the main hub screen, simply type the word nexus on your keyboard. The screen will transition to the red-themed Vault. Type it again to exit.

Stealth Mode: Press ~ (tilde) at any time to fake a Google search page.

Changing the Hub Password: Open index.html in a code editor, locate const HUB_PASSWORD = "nexusgames";, and change it to whatever you like.
